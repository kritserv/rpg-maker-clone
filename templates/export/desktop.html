Desktop Export Instructions

Make sure you've backup your game project and/or commit to git before export your game.

1. Open game folder

2. Edit game_data/game_mode.txt:

  • pc

3. If you have previously exported to Android and/or Browser, ensure you delete the folder names 'bin', 'build' and hidden folder names '.buildozer' before proceeding. Otherwise, Pyinstaller might attempt to include unnecessary file.

4. Create Executable file.

  • Run pyinstaller --onefile --noconsole main.py
  • The created executable file will be inside dist folder, move it outside (to the root game folder) and try to run the game.
  • Finally, remove the following:
    • folder name 'src'
    • folder name 'build'
    • hidden folder name '.git' and .gitignore (if using git)
    • requirements.txt
    Then, Pack every file in root game folder to zip and shipping your game to any platform.
  • The executable file should be able to execute without python install on user's pc.
  • check documentation for more info.
