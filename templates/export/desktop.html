Desktop Export Instructions

1. Open your game folder

2. Edit game mode setting:

  • Open game_data/game_mode.txt and set it to pc

3. Clean up previous builds:

If you previously exported to Android or Browser, delete:

  • Folder named bin
  • Folder named build
  • Hidden folder named .buildozer

This prevents PyInstaller from including unnecessary files.

4. Create executable file:

  • Run: pyinstaller --onefile --noconsole main.py
  • Move the executable from the dist folder to your root game folder
  • Test the executable to ensure it works properly

5. Prepare for distribution:

  • Remove the following items:
    • Folder named src
    • Folder named build
    • Hidden folder named .git and .gitignore (if using git)
    • requirements.txt
  • Zip all remaining files in the root game folder
  • The executable should run on users' PCs without Python installed

For more information, see the PyInstaller documentation
