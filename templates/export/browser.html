Browser Export Instructions

1. Open your game folder

2. Edit game mode setting:

  • Open game_data/game_mode.txt and set it to web

3. Clean up previous builds:

If you previously exported to Android or Desktop, delete:

  • Folder named bin
  • Folder named build
  • Hidden folder named .buildozer

This prevents Pygbag from including binary files alongside WebAssembly, which can cause errors.

4. Build and run:

  • For local testing, run: pygbag .
  • For deployment package, run: pygbag --archive .
    (Creates a compressed file that can be hosted on any static website like GitHub Pages or itch.io)

5. Test locally:

  • Open http://localhost:8000/ or
  • Open http://localhost:8000/?-i

For more information, see the Pygbag documentation
