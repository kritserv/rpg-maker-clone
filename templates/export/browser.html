Browser Export Instructions

1. Open game folder

2. Edit game_data/game_mode.txt:

  • web

3. If you have previously exported to Android, ensure you delete the folder names 'bin' and hidden folder names '.buildozer' before proceeding. Otherwise, Pygbag might attempt to include the binary file alongside WebAssembly, causing an error.

4. Build and Run:

  • Run pygbag . for test locally or pygbag --archive . for a compressed file which can be host on any static website such as github page or itch.io.
  • The process may take some time initially, but subsequent builds will be faster.
  • Test on http://localhost:8000/
  • or http://localhost:8000/?-i
  • check documentation for more info.
