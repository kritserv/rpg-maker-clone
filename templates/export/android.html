Android Build Instructions

1. Open game folder

2. Edit game_data/game_mode.txt:

  • android

3. If you have previously exported to Browser, ensure you delete the folder name 'build' before proceeding. Otherwise, Buildozer might attempt to include the browser file alongside, causing an error.

4. Install Buildozer readthedocs

  • sudo apt update -y
  • sudo apt install -y git zip unzip openjdk-17-jdk python3-pip autoconf libtool pkg-config zlib1g-dev libncurses5-dev libncursesw5-dev libtinfo5 cmake libffi-dev libssl-dev

5. Initialize Buildozer:

  • Run buildozer init

6. Modify the following parameters in buildozer.spec:

  • source.include_exts =
  • requirements = python3==3.10.12, kivy==2.3.0, hostpython3==3.10.12, pygame, pyjnius, jnius, sdl2, sdl2_image, sdl2_mixer, sdl2_ttf, png, jpeg, pillow
  • presplash.filename = %(source.dir)s/android_presplash.png
  • icon.filename = %(source.dir)s/android_icon.png
  • orientation = landscape
  • fullscreen = 1
  • android.presplash_color = black
  • android.permissions = android.permission.READ_EXTERNAL_STORAGE, android.permission.WRITE_EXTERNAL_STORAGE, android.permission.MANAGE_EXTERNAL_STORAGE, android.permission.READ_PHONE_STATE

7. Build and Run:

  • Run buildozer android debug deploy run logcat
  • The first time you build with Buildozer, accept any required licenses.
  • The process may take some time initially, but subsequent builds will be faster.
